# 🎨 CANVA SETUP GUIDE - ATRIA PITCH
## Para importar y personalizar tus slides (3 minutos)

---

## 📥 **CÓMO IMPORTAR EL PPTX A CANVA**

1. Ve a **Canva.com** → Click en **"Create a design"**
2. Busca **"Presentation"** → Elige tamaño **"16:9"**
3. En el menú izquierdo, busca **"Upload"** → Selecciona `ATRIA_Pitch_Canva.pptx`
4. Canva convertirá automáticamente los slides ✅

## 🔧 **TIPS PARA CANVA**

1. **Importa el PPTX pero personaliza en Canva**
   - Los colores pueden necesitar ajuste
   - Las fuentes pueden cambiar (Cambria → Georgia)
   - Canva es muy flexible, aprovéchalo

2. **Para la SLIDE 3 (Demo)**
   - Haz el area de demo GRANDE
   - Pon instrucciones pequeñas abajo
   - Si prefieres, reemplaza el placeholder con una captura de tu MVP

3. **Descargas PDF para presentar**
   - Canva → **Download** → **PDF**
   - Abre en **Google Slides** (no PowerPoint) para screenshare smooth
   - O usa Canva en full screen mode

4. **Alternativa: Haz todo directo en Canva**
   - Si prefieres, ignora el PPTX
   - Crea cada slide manualmente en Canva
   - Usa esta guía como template
