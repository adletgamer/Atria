# Project Mission

This project is not a blockchain demo.
It is a decision-state infrastructure for regulated perishable trade.

## Primary Product Question

"Is this consignment decision-ready for import/compliance and financing review?"

## Core Outputs

- Computed state (evidence completeness, attribution strength, custody continuity)
- Blocking exceptions
- Evidence completeness score
- Evidence pack (verifiable, hashable, shareable)
- Verifiable anchor (blockchain commitment of hash)

## Do Not Optimize For

- Consumer scan
- Marketplace
- Wallet UX
- Public decorative trust score
- Multi-crop expansion
